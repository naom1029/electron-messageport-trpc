This guide walks through a minimal setup where a **renderer** process calls procedures on the **main** process.

<Steps>

1. **Define your tRPC router (main process)**

    ```typescript
    // src/main/router.ts
    import { initTRPC } from '@trpc/server';
    import { z } from 'zod';

    const t = initTRPC.create();

    export const appRouter = t.router({
      greet: t.procedure
        .input(z.object({ name: z.string() }))
        .query(({ input }) => {
          return `Hello, ${input.name}!`;
        }),
    });

    export type AppRouter = typeof appRouter;
    ```

2. **Define the Electron tRPC channels**

    ```typescript
    // src/main/trpc.ts
    import { defineElectronTRPC } from 'electron-messageport-trpc';
    import type { AppRouter } from './router';

    export const electronTRPC = defineElectronTRPC<{
      main: AppRouter;
    }>();
    ```

3. **Attach your router to the window (main process)**

    ```typescript
    // src/main/index.ts
    import { app, BrowserWindow } from 'electron';
    import { createElectronTRPCMain } from 'electron-messageport-trpc/main';
    import { appRouter } from './router';
    import { electronTRPC } from './trpc';

    app.whenReady().then(() => {
      const win = new BrowserWindow({
        webPreferences: {
          preload: '/path/to/preload.js',
        },
      });

      createElectronTRPCMain({
        channels: electronTRPC,
        routers: {
          main: appRouter,
        },
        windows: [win],
      });

      win.loadFile('index.html');
    });
    ```

4. **Expose the tRPC bridge (preload script)**

    ```typescript
    // src/preload/index.ts
    import { exposeElectronTRPC } from 'electron-messageport-trpc/preload';
    import { electronTRPC } from '../main/trpc';

    exposeElectronTRPC(electronTRPC);
    ```

5. **Create the tRPC client (renderer process)**

    ```typescript
    // src/renderer/index.ts
    import { createElectronTRPCClient } from 'electron-messageport-trpc/renderer';
    import { electronTRPC } from '../main/trpc';

    const trpc = createElectronTRPCClient(electronTRPC);

    const greeting = await trpc.main.greet.query({ name: 'World' });
    console.log(greeting); // "Hello, World!"
    ```

</Steps>

## What Just Happened?

1. `defineElectronTRPC()` creates a typed channel contract shared by main, preload, and renderer code.
2. `createElectronTRPCMain()` connects the `main` channel to `appRouter` for each window.
3. `exposeElectronTRPC()` receives transferred MessagePorts in preload and exposes the bridge to the renderer.
4. `createElectronTRPCClient()` creates normal tRPC clients for each channel. The MessagePort setup stays inside the Electron helpers.

<Aside type="tip">
The low-level exports are still available if you want custom lifecycle control or direct MessagePort access.
</Aside>
